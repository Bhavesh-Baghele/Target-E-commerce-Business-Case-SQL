# Target-E-commerce-Business-Case-SQL

## Overview
This project analyzes customer orders, payments, logistics and delivery performance of an e-commerce company using SQL.

## Skills Demonstrated
- Advanced SQL (CTEs, Window Functions, Joins)
- Time series & seasonality analysis
- Customer & regional analysis
- Logistics and delivery performance analysis
- Business insight generation
